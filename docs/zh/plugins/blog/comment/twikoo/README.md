# Twikoo

一个简洁、安全、免费的静态网站评论系统，基于 [腾讯云开发](https://curl.qcloud.com/KnnJtUom)。

<!-- more -->

## 安装

```bash
npm i -D twikoo
```

## 快速上手

先按照 [官方文档](https://twikoo.js.org/backend.html) 部署好后端。

## 配置

请配置 `provider: "Twikoo"` 并将你的服务端地址传入插件选项中的 `envId`。

其他的配置项详见 [Twikoo 配置](config.md)。
