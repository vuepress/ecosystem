# theme-default

## Usage

Install `@vuepress/theme-default` :

```bash
npm install @vuepress/theme-default@next
```

```ts title=".vuepress/config.ts"
import { defaultTheme } from '@vuepress/theme-default'

export default {
  theme: defaultTheme({
    // set theme config here
  }),
}
```
