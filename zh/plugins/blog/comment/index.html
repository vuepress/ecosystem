comment

VuePress 评论插件，支持多种评论服务。

使用

npm i -D @vuepress/plugin-comment@next
.vuepress/config.ts
import { commentPlugin } from '@vuepress/plugin-comment'

export default {
  plugins: [
    commentPlugin({
      provider: 'Waline', // Artalk | Giscus | Waline | Twikoo
      // 服务商配置
    }),
  ],
}

支持的服务商

  • Artalk
  • Giscus
  • Twikoo
  • Waline
