默认主题

使用方法

安装默认主题：

npm i -D @vuepress/theme-default@next

在配置文件中指定主题：

.vuepress/config.ts
import { defaultTheme } from '@vuepress/theme-default'

export default {
  theme: defaultTheme({
    // 在这里添加主题配置
  }),
}
